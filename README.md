# Python Fast Api

Python Fast API base project

## Setup

Environment setup
```bash
python -m venv venv
source/venv/bin/activate
pip install -r requirements.txt
```

## Running the Project with Uvicorn
```bash
uvicorn src.main:app --reload
```
